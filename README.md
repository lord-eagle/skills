# skills

Private collection of Claude Code skills.

## Skills

- **review-triage** — Triage PR review feedback into Must fix / Should fix / Nice to have / Nit; bundles non-blocking findings into one GitHub issue per severity.
- **solo-orchestration** — Orchestrate Solo MCP spawned agents with push-based reporting instead of polling.
- **solo-multi-agent-brainstorm** — Moderated panel discussion across multiple Solo MCP projects and AI agents; spawns 3 panelists, runs N rounds with synthesis, delivers verdict.

## Layout

Each skill lives at the repo root with its `SKILL.md` and a `versions/` history.
